# Swift Clicks

<img src="logo.png" width="150" alt="the mod's logo" />

Now it is super easy to click twice (or more!) in one frame. For example, a multi-activate dash orb can follow the player, which normally means you cant jump. With this mod enabled, you click and it clicks twice in one frame, allowing you to jump.

Enable this mod and play level ID: 134553188
